UniqueJS30

What is UniqueJS30?

Hey there everyone, my name is David Nugent. I'm a developer based out of BC, Canada. At the end of July 2021 I decided to take on a project to develop an app a day for 30 days.

This was in-part inspired by Wes Bos' JS30 free course - which I would highly recommend for any junior dev's, you can find it here. In this course, the idea is to learn by building and to only use vanillaJS with no packages or frameworks. As an avid user of frameworks such as React I had some trepidation and looking at the course at a glance I wasn't too sure I would actually learn any new skills, so I decided to take it one step further by building upon the daily concepts and build something unique every day instead. Thus UniqueJS30 was born.

Below, as well as videos and descriptions, you can find live demos and source code. The sites below were originally built for desktop viewing but I will make these designs adaptive at a later stage. If you enjoy the concept or any of the projects, feel free to connect with me directly here.

Projects

01

Piano Emulator ⛛

For day one, I created a piano emulator. The piano emulator uses keycode detection on keyboard presses and maps these to specific audio elements which are subsequently played allowing a user to emulate the experience of pressing a key on a piano. In addition I added some dynamic transition effects to give the visual appearance of key presses.

Live DEMO Github

02

Timezone Clock ⛛

For day two, I created an app which displays css-generated clocks that imitate analogue clocks. It also has the ability to generate clocks from different regions which can account for daylight savings. I had a lot of fun creating a factory function to store and update each clocks logic.

Live DEMO Github

03

Photo Editor ⛛

For day 3, I created a photo editor. The app utilizes the canvas API combined with filters to allow users to upload a photo, edit it and download the result.

Live DEMO Github

04

JavaScript W[arr]ior ⛛

For day four, I created a game which allows you to practice array methods through fighting a horde of monsters. The app uses dynamic form fields combined with an imitation testing library to test user input.

Live DEMO Github

05

Flex Calendar ⛛

For day five, I explored creative uses of flex-grow and used it to create a dynamic calendar/day-planner. I also explored using cubic-beziers in order to create timing transitions for the flex-adaption.

Live DEMO Github

06

Nuge Docs ⛛

For day six, I created a simple version of an online text-editor. Utilizing the Words API tp create a dynamic lookup for definitions, synonyms, examples and rhymes.

Live DEMO Github

07

JavaScript W[arr]ior 2 ⛛

For day seven, I revisited the javascript game I had made earlier for practicing array methods. I developed the idea further and made it into a retro game where you can control the actions of a character and move them around a map followed by object-orientated multi-question challenges. Of particular note are the coordinate driven controls for the map.

Live DEMO Github

08

sketchPad ⛛

For day eight, I created a digital painting tool that utilizes the canvas API. For this app there was a lot on mouse events and coordiante control. There are a number of your atypical painting tools as well as the ability to save your creations. Finally, an undo and redo functionality was added.
